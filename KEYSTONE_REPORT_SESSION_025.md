## 6. Methodology note (one improvement)

graph.json can silently drift from the code the same way lint did before
the S024 CI gate. **Add a CI step that re-runs `memory/ast_graph.py` and
fails on `git diff --exit-code graph.json`**, so the dependency graph is
mechanically guaranteed to match the source on every push — the same
"enforce the invariant in CI, not by hand" pattern that fixed lint drift.
